The problem: I wanted to work on my D3 skills and try out a loom graph visualization. I also wanted to create a visualization that was related to animals or the environment.

The solution: I found an article by National Geographic on the tools that animals use and wanted to create a visualization that was more integrated and interactive.

Take a look inside here!

This was my first experience coding a loom graph, and I adapted my code from Nadieh Bremerâ€™s blog post on loom graphs. What I liked about this graph was the ability to not only visualize the number of tools that each animal could use but also which tools were more common than others. This was just a personal project to help enhance my D3 skills and data visualization design. The biggest challenges I encountered were determining how to adjust the graph to include more variables, as the existing code had about half the number attributes in the middle. This required me to dive into the source code and adjust the distance between each attribute and also the length and curve of the chords themselves.

Given the high level of user interaction, I wanted to keep the design clean and straightforward. I landed on an orange and blue theme because I felt these were Earthy colors that combined land and water in a form that was readable and simple. I also drew each of the animal icons in Adobe Fresco. I felt that the small animal icons that are visible on hover allow for an enhanced visual experience without distracting from the data itself.
