The problem: With occupancy regulations due to the pandemic, my local climbing gym started using a reservation system for climbing sessions. I found the reservation system to be quite frustrating and generic and wanted to design not only a new reservation system but visual identity all together.

The solution: I started with redesigning their logo and brand language. I then applied that language to a mobile application that would allow climbers to first and foremost reserve climbing sessions as well as other information.
