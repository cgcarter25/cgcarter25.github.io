The problem: I wanted to work on my D3 skills and create an interactive dog breed visualization. As a dog lover, I've seen many infographics about various breeds but never a complete visualization on the data collected by the AKC.

The solution: Using AKC data, I mined and cleaned the data in Excel and then played with it in D3 to create the right look. The doodle of the dog was a last minute addition to add a little bit of texture to the piece.

I loved creating this project because it came together as I went along. I ended up only being able to find data that I could easily extract from the AKCâ€™s website for the past 20 years or so and then I had to manually input the other years from scanned logs that had been hand-written. I am especially pleased with the user-interaction of this visualization. I ended up needing to overlay an invisible scatterplot and Voronoi diagram to create the hover effect that would allow the user to see the each breed-line. This was tricky because simply using a mouse-over effect on the line graph made it so that the user needed their cursor over a very narrow part of the line. The Voronoi diagram allows for the user to hover in the space that is closest to each cell denoting a specific rank and year.

I chose an orange and teal color palette because they felt like friendly, non-descript colors. I thought about using a more neutral furry palette but felt like it was too depressing and that a visualization about dogs needed to be fun and playful. This is also why I ended up including the doggy doodle in the top right. I still plan on updating this visualization, and perhaps using line-weight or color to denote average rank of the breed or highest rank achieved but have not had the time to do so.
